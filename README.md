## Usage

* Install requirements
```bash
$ poetry install
```
* Run the module
```bash
$ poetry run python3 -m stella
```

## Controls

* WSAD - fly forward/backwards/left/right
* Arrow keys left/right - rotate left/right
* Arrow keys up/down - fly up/down
* Space - takeoff
* Enter - land
* O/P - decrease/increase speed
* F12 - save photo
